TEKNO PHANTOM AGENT - RUN COMMANDS (WINDOWS)
============================================

Open 3 terminals and run each service separately.

0) ENV SETUP (ROOT .env)
------------------------
Update root .env with these values before starting services:

LLM_MODE=cloud
CLOUD_PROVIDER=openai
OPENAI_API_KEY=your_openai_key_here
OPENAI_MODEL=gpt-4.1-mini
BROWSER_MODE=playwright
PLAYWRIGHT_HEADLESS=false
PLAYWRIGHT_BROWSER=chromium

1) BRAIN (LLM service)
----------------------
PowerShell:
cd "C:\Users\Teknotrait Solutions\Desktop\Tekno-Phatom-agent\brain"
if (!(Test-Path .venv)) { python -m venv .venv }
.\.venv\Scripts\python.exe -m pip install -e .
.\.venv\Scripts\python.exe -m uvicorn app.main:app --reload --host 127.0.0.1 --port 8090

CMD:
cd /d "C:\Users\Teknotrait Solutions\Desktop\Tekno-Phatom-agent\brain"
if not exist .venv python -m venv .venv
.\.venv\Scripts\python.exe -m pip install -e .
.\.venv\Scripts\python.exe -m uvicorn app.main:app --reload --host 127.0.0.1 --port 8090

Health check:
http://127.0.0.1:8090/health
